== Description ==

InLinkz LinkUp is a widget that is inserted in your wordpress blog and allows you to receive link submissions from your readers.
You are in total control of when the widget is available, how it is displayed and who is notified in the event of a submission.

== Installation ==

*Using the WordPress dashboard*

1. Login to your weblog
1. Go to Plugins
1. Select Add New
1. Search for *inLinkz LinkUp*
1. Select Install
1. Select Install Now
1. Select Activate Plugin

*Manual*

1. Download and unzip the plugin
1. Upload the entire inlinkz-linkup/ directory to the /wp-content/plugins/ directory
1. Activate the plugin through the Plugins menu in WordPress
